### 연산자

#### 단항 연산자

- 증감연산자 : ++ , --
  피연산자의 값에 1을 더하거나 빼는 연산자

  - 전위연산자 ++a
    먼저 연산 후 다른 연산 실행
  - 후위연산자 a++
    다른 연산 우선 실행 후 연산

- 논리 부정 연산자 : !
  논리값을 부정
  제어문 활용시 사용

        boolean bool1 = true;
        boolean bool2 = !bool1;
        System.out.println(bool2);

        =
        false

- 산술 연산자 : \*, /, %

  - '\*' 곱한값
  - '/' 나눈 값
  - '%' 나머지 값
  - '+' 더한값
  - '-' 뺀값

- 비교 연산자 : == , !=

  - a == b 같으면 true
  - a != b 같지 않으면 true

- 논리 연산자 : && , ||
  값을 비교하는 연산자

  - a && b a and b
  - a || b a or b

- 복합 대입 연산자 : +=, -+, \*=, /=, %=
  다른 연산자와 대입연산자가 결합

        a += 10 === a = a + 10
        a -= 10 === a = a - 10
        a *= 10 === a = a * 10
        a /= 10 === a = a / 10
        a $= 10 === a = a % 10

  - 증감 연산자(++,--) 는 1씩 증가 또는 감소
  - 복합 대입 연산자는 원하는 값을 연산자에 따라 결과값 도출

- 삼항 연산자 : 조건식 ? 식1 : 식2
  조건식이 true 면 식1 false 이면 식2

        int result1 = a > b? a++ : b--;
            result1 값은 a 또는 b 값
            이후 a, b 값을 실행시
            a가 b보다 크다면 a+1, 아닐경우 b-1;

        int a = 3;
        int b = 4;
        int result1 = a > b? a++ : b--;
        System.out.println("result1 : " + result1); == 4
        System.out.println("a : " + a); == 3
        System.out.println("b : " + b); == 3

- 중복 삼항 연산자 : 조건식1 ? 식1 : (조건식2 ? 식2 : 식3 );

        int result2 = c < d ? c++ : (d == 0 ? c-- : d++ );
        result2 값은 c 또는 d 값이며
        c, d 값을 실행시
        d가 c보다 크면 c++
        아닐 경우 d가 0이면 c-- , 아닐경우 d++

        int c = 3;
        int d = 4;
        int result2 = c < d ? c++ : (d == 0 ? c-- : d++ );
        System.out.println("result2 : " + result2); ==  3
        System.out.println("c : " + c); == 4
        System.out.println("d : " + d); == 4
